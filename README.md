# Fraud Detection Data Analytics Project

An end-to-end data analytics project analyzing transaction data to identify patterns and indicators of fraudulent activity. Built as part of the Naan Mudhalvan program, using a complete analytics pipeline from raw data to visual dashboard.

## Overview

This project takes raw transaction data and works through it using four different tools, each suited to a different stage of analysis:

- **Excel** — Initial data exploration, cleaning, and pivot table summaries
- **SQL** — Structured querying to identify fraud patterns and aggregate transaction insights
- **Python (Jupyter Notebook)** — Data analysis and exploration using pandas
- **Power BI** — Interactive dashboard visualizing fraud trends and key metrics
